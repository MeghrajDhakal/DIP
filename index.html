IMAGE ACQUISITION
The Gateway Step
Image acquisition is the first step in digital image processing. It involves capturing an image using a camera or sensor.

The captured image is converted to digital form through two key operations:
• Sampling — discretizing the spatial coordinates (x,y)
• Quantization — discretizing the intensity/amplitude values

IMAGE ENHANCEMENT
Making it Beautiful (Subjective)
Enhancement improves visual quality for human interpretation. It's a subjective process — results depend on what looks good to the viewer.

Common methods: • Contrast Stretching — expands the intensity range
• Image Sharpening — enhances edges
• Histogram Equalization — redistributes intensity values

Ex: Improving brightness & contrast of a dark photograph.

IMAGE RESTORATION
Recover the Truth (Objective)
Restoration aims to recover the original image from a degraded version. It is objective — there is a known target state to restore to.

• Uses mathematical & statistical models to remove distortions
• Common degradations: Blur (defocus, motion) and Noise
• Unlike enhancement, restoration has a ground-truth goal

Example of degradation: defocus, motion blur.

COLOUR IMAGE PROCESSING
The World in Colour
Colour processing handles images containing colour information.

Key colour models:
• RGB — Red, Green, Blue (additive, used in screens)
• CMY — Cyan, Magenta, Yellow (subtractive, used in printing)

Widely used in graphics, photography, and video processing. Each colour channel can be processed independently or jointly.

IMAGE COMPRESSION
Less Space, More Speed
Compression reduces data size required to represent an image — saves storage and improves transmission efficiency.

Two fundamental types:
• Lossy (JPEG) — Data is discarded; higher compression but quality loss
• Lossless (PNG, DIVC) — No information lost; perfect reconstruction

Trade-off: File size ↓ vs. Image quality ↑

IMAGE SEGMENTATION
Divide & Conquer
Segmentation divides an image into meaningful regions or objects. Simplifies image analysis by separating objects from the background.

• Enables focused analysis of specific image parts
• Critical in medical imaging (tumour detection)
• Used in autonomous driving, surveillance

Ex: Separating a tumour region from a medical scan.

OBJECT RECOGNITION
Machines That See
Object recognition identifies and labels objects in an image based on extracted features.

Process:
1. Feature Extraction — edges, shapes, textures
2. Pattern Matching — compare with stored database
3. Classification — assign a label/category

Examples: Face recognition, fingerprint identification, etc.

SPATIAL DOMAIN TRANSFORMS

T1
SPATIAL DOMAIN
Work directly on image pixels f(x,y)
In spatial domain, operations are performed directly on pixel values.
Two main categories:
• Intensity Transformation — operates on single pixels → s = T(r)
• Spatial Filtering — operates on a neighbourhood of pixels
g(x,y) = T[ f(x,y) ]    where T = transformation operator
T2
FREQUENCY DOMAIN
Transform → Filter → Inverse Transform
In frequency domain, the image is first transformed (e.g., via Fourier Transform), filtered in the transform domain, then inverse-transformed back to spatial domain.

Flow: Spatial Image → Transform → Filter in Freq. Domain → Inverse Transform → Output
FREQUENCY DOMAIN PIPELINE
LINEAR
s = r (Identity)
s = L-1-r (Negative)
Straight-line mappings. Identity leaves image unchanged; Negative inverts intensities.
LOGARITHMIC
s = c·log(1+r)
Compresses bright, expands dark. Useful for Fourier spectrum display. Inverse log does the opposite.
POWER LAW (γ)
s = c·rγ
Gamma correction. γ<1 brightens; γ>1 darkens. Used in display calibration (CRT screens).
T3
THRESHOLDING
Creating Binary Images
Image thresholding converts a grayscale image to binary (black/white):
• If r ≤ K → s = 0 (Black)
• If r > K → s = L-1 (White)

The threshold value K determines the cut-off point.
s = 0 if r ≤ K   |   s = L-1 if r > K
Threshold K K=128
